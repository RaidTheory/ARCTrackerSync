# Contributing to ARCTracker Sync

Thanks for your interest in improving ARCTracker Sync. Bug reports, feature
ideas, and pull requests are all welcome.

## Reporting issues

- Use GitHub Issues for bugs and feature requests.
- For security vulnerabilities, follow [`SECURITY.md`](SECURITY.md) instead of
  opening a public issue.

## Development

```powershell
cargo build --release
cargo run
```

See the [README](README.md#build-from-source) for prerequisites (Rust toolchain
and Windows). Please run `cargo fmt` and `cargo clippy` before submitting, and
keep pull requests focused.
